This project is a console-based Text Adventure Game developed using C++. The game allows players to explore a virtual world through text commands, make decisions, and interact with the environment. It demonstrates core programming concepts such as control structures, functions, classes, and file handling.

The game follows a storyline where the player navigates through different locations, solves puzzles, and encounters challenges. Each decision taken by the player affects the outcome of the game, making it interactive and engaging. The project focuses on enhancing logical thinking, problem-solving skills, and understanding of object-oriented programming in C++.

🎮 Features
Interactive story-based gameplay
Multiple locations and paths
Decision-making system (choices affect outcome)
Inventory management system
Simple command-based input (e.g., go, take, use)
Win/Lose conditions
🛠️ Technologies Used
Language: C++
Platform: Console / Terminal
Concepts Used:
Classes and Objects
Functions
Conditional Statements (if-else, switch)
Loops (while, for)
File Handling (optional for saving progress)

🚀 How to Run
Install a C++ compiler (like g++)

Compile the program:

g++ main.cpp game.cpp player.cpp rooms.cpp -o game

Run the executable:

./game
🎯 Gameplay Instructions
Enter commands to play (example: go north, take key, use potion)
Explore different rooms and collect items
Solve puzzles to progress
Reach the final goal to win the game
